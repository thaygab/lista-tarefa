# 📝 Lista de Tarefas

> Uma aplicação web desenvolvida com **HTML, CSS e JavaScript puro**, criada para praticar manipulação do DOM, organização de código e persistência de dados utilizando Local Storage.

![Preview do projeto](./assets/preview.png)

---

## ✨ Funcionalidades

- ✅ Adicionar tarefas
- ✏️ Editar tarefas
- 🗑️ Excluir tarefas
- ✔️ Marcar tarefas como concluídas
- 📅 Definir prazo
- 🏷️ Prioridade (Baixa, Média e Alta)
- 📊 Dashboard com estatísticas
- 🔍 Filtros por status
- 💾 Salvamento automático com Local Storage
- 📱 Layout responsivo

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Local Storage
- CSS Grid
- Flexbox
- Git & GitHub

## 📚 Conceitos praticados

Durante o desenvolvimento deste projeto foram praticados conceitos importantes do desenvolvimento Front-end, como:

- Manipulação do DOM
- Eventos
- Local Storage
- Arrays
- map()
- filter()
- find()
- JSON.parse()
- JSON.stringify()
- CSS Grid
- Flexbox
- Responsividade
- Organização e refatoração de código
